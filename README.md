# 🌾 Agro Vivo — Agricultura do Futuro

Projeto educativo desenvolvido em HTML, CSS e JavaScript sobre o agronegócio moderno e sua importância para o mundo.

---

## 🚜 Sobre o projeto

O **Agro Vivo** mostra como a agricultura evoluiu de práticas simples para um sistema altamente tecnológico e essencial para a vida humana.

---

## 🌍 Temas abordados

- Agronegócio moderno
- Tecnologia no campo
- Sustentabilidade ambiental
- Produção global de alimentos
- Impacto econômico do agro

---

## 🧠 Objetivo

Mostrar a importância do agro na sociedade moderna de forma visual, educativa e interativa.

---

## ⚙️ Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- Imagens do Unsplash
- Google Fonts

---

## ✨ Diferenciais

✔ Design moderno e vivo  
✔ Imagens grandes e chamativas  
✔ Animações ao rolar  
✔ Conteúdo educativo completo  
✔ Estilo de site institucional
